List out the different ways to select an item

  • Select an element simply by name (ex: "p")
  • Select by the ID name (ex: "#myID")
  • Select all elements with the given class (ex: ".myClass"). Multiple classes can be with a space (ex: "class1 class2")
  • You can even select only tags with a specific tag (ex: "p.myClass")
  • * is the universal selector, which affets all HTML elements.
  • You can apply CSS styling to multiple elements (ex: "h3, h4 {...}")
  • A descendant selector applies to all matching elements inside a specific element (ex: "div p {...}"")
  • Psuedo classes use a colon to refer to an element's special state (ex: "a:hover")

What is the difference between an ID and a Class?

ID's are used as an attribute for a specific tag (i.e. one only), whereas classes can apply for multiple different tags.

In practice, ID's are typically a noun, whereas classes are usually an adjective describing what the class does.

What is the difference between a SPAN tag and a DIV tag

Both a span and a div tag can be used for CSS styling, but div tags are used for grouping elements of a webpage together. The span tag is typically for inline styling.

What is the difference between a DIV tag and a HEADER, MAIN, or FOOTER?

The div tag, unlike header, main, and footer, is non-semantic, meaning it is not recognized as a meaningful divison of the website by crawlers.

What are the different ways to select a color?

  • Hexadecimal (ex: "#00FFA2")
  • RGB (ex: "rgb(100,60,90)")
  • RGBA (ex: "rgb(100,60,90,0.7)" where A is the alpha or transparency)
  • Name (ex: "cyan")
  • HSL (hue, saturation, lighness) (ex: "hsl(115, 50%, 60%))
