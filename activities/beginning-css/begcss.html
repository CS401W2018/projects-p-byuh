Inline CSS

While not technically wrong, is not best practice. Inline styles are specfically applied to a tag or element. This is done using the 'style' attribute in a tag opening.

Embedded CSS

The second order of priority, embedded CSS uses a <style> tag inside the head. This applies the CSS styling solely within the wepage.

External CSS

The least prioritized, external CSS references a .css file outside the HTML file.

You should use external CSS because this allows for multiple pages to reference the same styling configuration for a consistent theme, and easier coordination for a development team.

What is a selector used for?

Selectors are used in CSS to specify which tag to apply the styling to. You can apply a selector a type (aka type selector), such as header or paragraph tags. You can also use a class selector, which will apply the styling to all tags with the given class.
