Explain the difference between an absolute link and a relative link. Why would you use relative links?

An absolute link contains the Protocol (https://), Domain (byuh.edu), and Path (/cs-140). A relative only has the path.

You use absolute link when linking to a webpage outside of your domain. You use a relative link when linking to a page within your website/domain, for example: Profile Page

List the 3 rules with relative links.

  1. Same folder: "file.html"
  2. Child folder: "folder/file.html"
  3. Up: "../file.html"

Do links always need to be inside a NAV tag?

No. Links can be anywhere on the page. The NAV tag are used for links that help navigate the user through your website. Ideally, the nav is consistent between all pages.
