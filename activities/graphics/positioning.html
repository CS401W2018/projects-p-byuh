Engagement Questions

1. What is normal flow in HTML?

Normal flow is the default relative position of your web page elements before any styling is applied. Basically, the position is determined by its location relative to other elements within the HTML file.

2. What does float do? Why do you need to clear the float?

Float removes an element from the normal HTML flow, meaning the layout or position of an element is no longer dependent on its location relative to to other elements. For example, the flower image can be positioned to the right of this text body instead of only afterwards.

In order to clear the float, use the ''clear'' styling command assigned to a class.

3. What do absolute, fixed, and relative do?

  • Absolute: the postion of an element is not within the normal flow, but is assigned to a specifically defined area of the page.
  • Fixed: This works similarly to absolute, except the positioning of the element is defined to an area of the user's viewport instead of the page. For example, a nav bar can be fixed to always show at the top of the screen even when scrolling throughout a webpage.
  • Relative: changes to an elements relative position are applied after is current, normal flow position. For example, you can position an element 5px right of it's current, in flow position.

4. How can you make nice navigation buttons for mobile devices?

You can make nice navigation buttons for mobile devices by having a compact nav bar fixed to the top or side of the screen. You could also only have the navbar show when the user click a button (ex: ''hamburger menu'') in order to hide the navbar when not needed, but have the button fixed.
