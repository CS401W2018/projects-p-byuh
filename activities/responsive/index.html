Engagement Questions Mar-5

1. Why do we need a meta viewport tag?

We use the veiwport tag to tell the browser how we want the webpage to be displayed depending on the device and dimensions of the user's screen.

2. When you select the same tag twice, and in one of them you say "flex-direction: column;" and the other says "flex-direction: row;", which one takes priority?

The one listed second (lower in the stylesheet) is the one that takes priority. This acts as the latest set of instructions for the browser.

3. How do you change the CSS for a web page displayed on a phone?

You can use a '@media' tag to indicate specific styling depending on the size of the display, such as the flex direction of elements or the layout of the navigation bar.

Engagement Questions Mar-3

1. Why is the viewport meta tag important?

The viewport tag helps the wep page's content display correctly on different devices. If this is missing, all responsive elements on your page won't work.

2. How can you adjust the CSS for printing?

Create another stylesheet dedicated for sytling the sheet for printing. Then add to the head a link to that css with the attribute ''media = 'print'''.

3. When would you want to adjust the CSS for printing?

You may want to use a more accessible or universal font family when printing, such as a font that is easier for reading on paper.

You can also hide unnecessary webpage elements such as the navigation bar.
