Engagement Questions

1. Why is the viewport meta tag important?

The viewport tag helps the wep page's content display correctly on different devices. If this is missing, all responsive elements on your page won't work.

2. How can you adjust the CSS for printing?

Create another stylesheet dedicated for sytling the sheet for printing. Then add to the head a link to that css with the attribute ''media = 'print'''.

3. When would you want to adjust the CSS for printing?

You may want to use a more accessible or universal font family when printing, such as a font that is easier for reading on paper.

You can also hide unnecessary webpage elements such as the navigation bar.
