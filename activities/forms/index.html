Engagement Questions

  1. The form tag has an "action" attribute. What does it do?

    The action attribute is where the data is sent (such as a processing.html page) when the user submits the form.

  2. The form tag has a "method" attribute. What does it do?

    The method attribute determines how the data is handled once the user submits with.

  3. What is the difference between a "POST" and a "GET"?

    We use a GET request when solely intend to read, or retrieve data from a server.

    We use a POST request when we are sending data to be stored on the server.
