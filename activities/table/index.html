Engagement Questions | Wednesday, March 12

1. Why don't we use the border attribute?

The border attribute has been deprecated in the favor of the more customizable CSS table styling.

2. What is the easiest way to color every other row a different color?

In addition to manually assigning a class to every other row, you more easily created a alternate color effect by using a psuedo class that applies the styling to every even row.

For example: tbody tr:nth-child(even) { background-color: gray }

Engagement Questions | Monday, March 10

1. When do you use a table?

You should use a table any time you are displaying data.

2. When do you not use a table?

You should not use tables as a method to stylize your web page, such as creating a sidebar.

3. When do you need to do extra work for table accessibility?

It's important to use the appropriate tags and labels when wanting to associate specific headers and subheads, footers, etc with their relevant data. This is important so accessibility tools can understand what data is associated with which label.
