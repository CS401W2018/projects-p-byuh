Internet

A connection of numerous public networks from around the world communicating through a standard protocol.

Web

The system of websites or pages that users can access on the internet using a web browser.

How are the Internet and Web different?

The internet is the means by how are all the different elements of the web (websites, email programs, etc.) are connected to eachother. In other words, the internet is the road the connects the different buildings, or websites.

ASCII and Unicode

Each are character sets that can be encoded into a binary format. Unicode has characters and support for several languages, with each character represented by up to 4 bytes.

ASCII (American Standard Code for Information Interchange) only uses one byte per character, but only has support for the Latin alphabet.

What is UTF-8

UTF is a version of Unicode with improvements and compatibility with ASCII.

URL

A Uniform Resoure Locator (URL), aka "web address" is director reference to a specific location on the internet.

Domain Name

The domain name is the unique identifier for a website (e.g. google.com), as opposed to address to a specific page.

HTTP

HyperText Transfer Protocol (HTTP) is the protocol typically used on the internet to exchange information such as images or pages.

HTTPS

HyperText Transfer Protocol Secure (HTTPS) Works similarly to HTTP, but communicates through a secure and encrypted process.

IP Address

The number address for a specific device on a network.

DNS

Domain Name System (DNS) is a server and service that takes a given domain name and returns the associated IP address for that domain name. DNS is like a internet phonebook, returning the address when given a name.

W3C Recommendations

Is a guide for best practices and webs standards that are used on the web to best optimize connected networks.

XML, XHTML, HTML, HTML5

eXtensible Markup Language (XML) is a basic, universal structure for storing and sending data to different computer systems, such as a list of product prices to a webstore server. It also supports custom tags unlike HTML.

eXtensible HTML (XHTML) can be described as "HTML in XML syntax", and designed to easily be used alongside XML. However, it does contain behaviors different from HTML.

HyperText Markup Language (HTML), compared to XMl, is a format specifically used for making content on web page that is meant to be shown in a web browser. An HTML document is comprised of elements, such as text and images, which are the content and metadata that makes up the web page. These elements are defined by different tags.

HTML5 is basically an updated version of HTML with more advanced, accessible, and modern features; such as better support for different media formats and more logical semantic structure.
